\subsection{Chess Interpretability}

\paragraph{Explaining chess models}
Chess has always been a good playground for AI, and explanability is no exception \cite{McGrath2022Acquisition}. Simplified versions of this game have even been created to make research easy \cite{hammersborg2023reinforcement,hammersborg2023information}.
It is even possible to explore planning, including tree search, through dynamical concepts \cite{schut2023bridging}.

\paragraph{Explainable tree search}
It is possible to make tree search explainable by default. By extracting a policy using a surrogate model \cite{Soemers2022ExtractingTL} or using a simpler heuristic model \cite{Soemers2019BiasingMW}.
